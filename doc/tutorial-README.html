README

Implementation require for XScript and client-side

Version: 1.6

Lisence: MIT

Features:

  • Поддержка выполнения модулей в браузере.
  • На серверной стороне загружает модуль по его namespace или абсолютному пути. Логика поиска файлов реализована в Module#load. Исходный код модуля выполняется один раз и далее отдается из внутреннего кеша.
  • Поддерживается регистрация модулей через метод require.define.
  • Реализует поддержку пространств имен модулей и сохраняет все загруженные модули в глобальном объекте $XM в соответствии с их Module#namespace.
  • Поддерживает уровни переопределения и наследование модулей через require.path.

За создание и инициализацию модуля отвечает конструктор Module.
